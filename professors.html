// How to Update

02 — EDIT_GUIDE
01
Find the Professor on RMP

Go to ratemyprofessors.com, search your professor's name and university. Copy their overall rating, difficulty, would-take-again %, and profile URL.

02
Edit professors.html

Open professors.html in VSCode. Find the PROFESSORS DATA object in the script tag at the bottom and fill in each field for the relevant card.

03
Push to GitHub

Save the file, then run git add ., git commit -m "update prof ratings", and git push. Vercel auto-deploys in ~30s.
