02 — C_CHEATSHEET
Pointer Declaration
int *p = &x;
Declares a pointer to int. &x gives the address of x. *p dereferences — gives the value at that address.
Dynamic Allocation
malloc(n * sizeof(T))
Allocates n elements of type T on the heap. Returns void* — always cast. Always free() when done.
Struct Access
s.field / p->field
Dot operator for struct value, arrow operator for struct pointer. p->field is shorthand for (*p).field.
String Copy
strcpy(dst, src)
Copies src into dst including null terminator. Use strncpy for bounded copy. Never forget to allocate enough space.
File Open
fopen(path, mode)
Modes: "r" read, "w" write (truncate), "a" append, "rb"/"wb" for binary. Returns NULL on failure — always check.
Array / Pointer
arr[i] == *(arr+i)
Array name decays to pointer to first element. arr[i] and *(arr+i) are identical — pointer arithmetic under the hood.
